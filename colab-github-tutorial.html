Getting Started with Google Colab and GitHub

Fengjiao Tu  ·  2024  ·  6 min read

In this course, we use Google Colab to write and run Python code, and GitHub to store and submit your work. This tutorial walks you through the full setup — from creating accounts to submitting your first assignment — step by step.

No prior experience with either tool is required. Just follow along and you'll be ready to go.

Before you start: You need a Google account (Gmail) and will create a free GitHub account. Both are free of charge.
1
Create a GitHub account (or sign in)

Go to https://github.com/ and sign up for a free account if you don't already have one. If you have an existing account, simply sign in.

Keep your login credentials handy — you'll need them in later steps.

Fig 1. GitHub sign-up / sign-in page
2
Create a new repository for this course

Once logged into GitHub, click the "New" button on the left side of your dashboard to create a new repository.

Fig 2. Click "New" on the left side of your GitHub dashboard

Name your repository using the following convention:

Repository naming format
Format:   YourFirstName_CourseCode_Semester
Examples:
 · Fengjiao_INFO5731_Fall2024
 · Fengjiao_INFO4670_Spring2026
 · Fengjiao_INFO4670_Summer2026
Fig 3. Fill in the repository name and settings
Fig 4. Set visibility to Public and click "Create repository"

Fill in the repository name, leave the visibility as Public, and click "Create repository". Your repository is now ready.

Fig 5. Your new repository has been created successfully
3
Open Google Colab

Go to Google Colab using your Google account:

https://colab.research.google.com/

Sign in with your Gmail if prompted.

4
Create your first notebook

In Colab, click File → New notebook in Drive to create a new .ipynb file.

Fig 6. File → New notebook in Drive

Rename the file by clicking the title at the top. Use a descriptive name, for example:

in_class_exercise_01   or   assignment_01

Fig 7. Click the title to rename your notebook

You can add code blocks by clicking "+ Code" and text/markdown blocks by clicking "+ Text".

To run your code, click Runtime → Run all, or press Ctrl + Enter on a single cell. Save your work with Ctrl + S.

Fig 8. Add code cells and run them using Runtime → Run
5
Save your notebook to GitHub

When you're ready to save your work to GitHub, click File → Save a copy in GitHub.

Fig 9. File → Save a copy in GitHub

A dialog will appear. Select the repository you created in Step 2, and set the file path (you can keep the default or organize by folder). Then click OK.

Fig 10. Select your course repository and set the file path

The first time you do this, GitHub will ask you to authorize Colab — click Authorize google-colab and follow the prompts.

Your notebook is now saved to your GitHub repository. You can repeat this step every time you want to update your submission.

Fig 11. Your notebook is now saved on GitHub
6
Copy the link and submit on Canvas

Go to your GitHub repository, navigate to the file you just saved, and copy the URL from your browser's address bar. It will look something like:

https://github.com/YourName/YourRepo/blob/main/in_class_exercise_01.ipynb

Fig 12. Copy the URL from your browser and paste it into Canvas

Paste this link in the Canvas submission box for the assignment.

Academic Integrity: Your GitHub repository is public, which means your classmates can see your code. If we find that you have copied someone else's work, you will receive a zero — no exceptions. Always submit your own original work.

You're all set!

Once you complete this setup once, submitting future assignments is easy: write code in Colab → save to GitHub → paste the link in Canvas.

If you run into any issues during setup, feel free to reach out via Canvas Messages or come to office hours. Good luck! 🎉
